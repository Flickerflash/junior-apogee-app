# Security Policy

## Supported Versions

| Version | Supported |
|---------|----------|
| latest  | ✅       |

## Reporting a Vulnerability

Do **not** open a public GitHub issue for security vulnerabilities.

Please email: **security@flickerflash.vercel.app** or message via [LinkedIn](https://www.linkedin.com/in/andrewhensel).

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

You will receive a response within **72 hours**.

## Scope
This policy covers the `junior-apogee-app` codebase and its dependencies.
